Student Accommodation Listing Platform

A full-stack web application that helps university students find off-campus accommodation while enabling landlords to manage property listings and administrators to moderate the platform.
Built as a real-world solution with authentication, role-based access control, and modular backend architecture.
What This Project Solves
Finding safe and affordable student accommodation is difficult and fragmented.
This platform centralizes accommodation listings, improves transparency, and adds moderation features to protect students from fraudulent listings.
Core Features

Students
Browse and filter accommodation listings
View detailed property information
Report suspicious listings

Landlords
Register and authenticate securely
Add, update, and manage property listings
Submit landlord verification requests

Admin
Review landlord verification requests
Moderate reported listings
Manage users and platform activity

Technical Highlights
Role-based authentication (Student, Landlord, Admin)
Modular PHP backend with clear separation of concerns
Secure form handling and input validation
Dynamic filtering and data retrieval from SQL database
Clean, responsive UI built with utility-first CSS
AJAX-based reporting system for better user experience
Tech Stack
PHP, SQL (MySQL), TypeScript/JavaScript, Tailwind CSS, XAMPP, Git, GitHub

Architecture Overview
Frontend: Tailwind CSS + JavaScript for responsive UI and interactions
Backend: PHP APIs handling authentication, listings, reports, and admin logic
Database: Relational SQL database for users, properties, reports, and verification records
Security: Session management, role checks, and input sanitization
